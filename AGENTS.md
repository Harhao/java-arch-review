# Java Server Architecture Review Skill

This repository contains the `java-server-arch-review` skill for AI coding agents.

## Installation

```bash
npx skills add Harhao/java-server-arch-review
```

## Skill

The skill provides a 19-dimension architecture review for Java/Spring Boot projects covering security, performance, code layering, testing, database design, and more.

See [SKILL.md](./SKILL.md) for the full skill definition and [references/](./references/) for detailed review rules.
